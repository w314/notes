# PL/SQL Constructs

## Operators

- `||` - concatenation
- `:=` - assignment
- logical
    - `AND`
    - `NOT`
    - `OR`

## Conditional Constructs

```sql
BEGIN
    IF <condition> THEN
        statements;
    ELSIF <condition> THEN
        statements;
    ELSE
        statements;
    END IF;
END;
```
